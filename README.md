# meechanism.github.io

Hi. I'm Mee.

This sandbox is statically a generated website made with [Jekyll](https://jekyllrb.com/), used for testing stuff!

## Development

1.  Install:
    ```sh
    $ bundle
    ```
2.  Spin up dev server:
    ```sh
    $ bundle exec jekyll serve
    ```
3.  Browse:
    ```sh
    http://localhost:4000
    ```
